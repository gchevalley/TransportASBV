Contrôle de la connexion Internet

L'application de gestion de transport a besoin d'un accès à l'Internet Public pour obtenir des précisions sur les  itinéraires et le calcul des Km pour la facturation. Un certains nombre de tests peuvent être exécutés pour contrôler cet accès ( vitesse  et qualité de service )

Cliquez sur le lien dans le tableau pour exécuter le test

Description ou contexte d'utilisation

Valider la vitesse de votre accès  Internet Public Test de  vitesse  ( Appuyer sur Begin Test )
PingTest.net Mesure Perte de paquets et Qualité de service pour différentes destinations.
Vitesse  pour différentes  destinations Sélectionner  une destination pour le test de performance de bande passante
Internet Speed Test Vitesse et graphiques  avec tableau résumé
Cablecom  Speed  Test Upload/Download Speed
Speed Test  ( Speakeasy ) Upload/Download Speed avec différentes destinations
Google Map Accès interactif aux  cartes de Google
