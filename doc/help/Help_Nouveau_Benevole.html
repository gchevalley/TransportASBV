Ajouter un nouveau chauffeur bénévole

 

Lorsque vous ajoutez un nouveau chauffeur bénévole la forme WEB suivante va s'afficher et les  indications ci-dessous vous aiderons a remplir les  différentes cases avant de soumettre votre requête.

 

 

Certaines informations supplémentaires sont disponibles pour le profile d'un chauffeur  sur l'écran suivant  (  Page Suivante )

 

Lorsque vous  modifiez un chauffeur bénévole existant  la forme WEB suivante va s'afficher et les  indications ci-dessous vous aiderons a remplir les  différentes cases avant de soumettre votre requête. Cette forme permet entre autre de mettre à jour les profils de disponibilité du chauffeur.
