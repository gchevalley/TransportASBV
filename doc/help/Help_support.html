Support de l'application de gestion de transport ASBV.

 

Si vous rencontrez une difficulté lors de l'utilisation de l'application de transport ayez la gentillesse de vérifier les points suivants avant de solliciter une assistance téléphonique pour résoudre votre problème.

  • Le PC  ( Tour Acer )  est enclenché  ( Lumière Bleue sur le sommet droit de la tour du PC )  ?
  • L'écran plat est sous tension ( Lumière ambrée sur le  bas a droite de l'écran ) ?
  • Le router NetGear 3G ( Connexion à Internet ) est sous tension ( Interrupteur noir à l'arrière du routeur ) ?
  • La clé USB vers la fenêtre a un témoin au dos du support de clé  qui est coloré et qui ne doit pas clignoter   ?
  • La souris et le clavier sont connectés ?
  • Votre écran affiche la page d'accueil de l'application ?
  • Le pointeur de la souris est visible et suit les mouvements de votre main ?
  • Si votre question est relative à l'usage d'une fonction avez-vous consulté le mode d'emploi pour essayer de trouver une explication pour votre problème?

Si toutes ces vérifications ont été faites et  que votre application vous affiche des  messages d'erreur inhabituels contacter la personne ci-dessous pour vous aider. Avant d'appeler préparez une brève description de votre problème et tenter de vous rappeler  quelles sont les  dernières manipulations que vous avez faites avec votre clavier et votre souris.
