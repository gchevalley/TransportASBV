Arrêt du système

Description ou contexte d'utilisation

1.-  Fermer la fenêtre applicative Utiliser la sélection X  dans le coin supérieur droit de la fenêtre
2.- Arrêtez Windows Utiliser l'icone Windows dans le coin inferieur droit du bureau
3.- Vérifier que la tour PC est éteinte La lumière bleue sur le sommet de la tour est éteinte
4.- Eteindre le router Netgear 3G  Appuyer sur l'interrupteur Noir derrière le routeur
5.- Laisser l'écran en mode veille
