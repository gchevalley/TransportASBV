Moyennes pondérées utilisées pour la suggestion de  chauffeur 

A ce stade  le système possède pour chaque critère et chauffeur potentiel une liste de notes qualificatives brutes  dont on va pondérer l'importance.  Cette pondération va attribuer a chaque  critère une importance relative qui dépend des choix définis par l'administrateur de la  filiale.  Par exemple la distance entre le domicile du chauffeur et du passager a le poids plus important dans l'attribution de la note finale.

 

Exemple de calcul  pour le premier chauffeur ( Symbolisée par la  Voiture Bleue )   :   

Note Brute ( Voir Normalisation )

Pondération  exprimée en %

Ajustable par  filiale voir ci dessous

Score  pondéré (  Notes cumulées )
80  ( Distance ) 35 80 * 0.35 = 28
50 ( Nombre de transport ) 5 50 * 0.05 =  2.5
100 ( A déjà transporté ) 25 100 * 0.25 = 25
80 ( Nombre de trajets ) 25 80 * 0.25 = 20
0  ( Km parcourus ) 10 0 *  0.1 = 0
0  ( Transport Locaux ) 20 0 * 0.2 = 0
310 points sur un total de  600

310 /600  = 0.5166 -> 52 /100 Moyenne non pondérée 

Total : 28 + 2.5 + 25  + 20 = 75.5 -> 76 / 100

 

Les paramètres peuvent être ajusté  pour chaque filiale (  Filiale ->Editer )

 

 

La liste des chauffeurs sera présentée sur votre écran dans l'ordre décroissant de la note finale  (  76  - 66 - 40 )

 

A ce stade  on obtient une liste des chauffeurs  pour ce transport qui est présentée sur votre  écran dans l'ordre d'attribution et il  suffira au permanencier d'appeler les chauffeurs  en tête de liste  jusqu'à l'obtention d'une réponse  positive. Des que l'accord du chauffeur est obtenu par téléphone le permanencier va attribuer le mandat de transport et la  requête de transport  va disparaitre de la liste  d'attente. Si le chauffeur le désire le système va lui communiquer tous les détails par courrier électronique y compris les informations d'itinéraire.
