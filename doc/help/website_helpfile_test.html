Procedures de sauvegarde  

Lorsque vous exécutez l'application de gestion de transport le système crée au démarrage  une sauvegarde totale qui peut être utilisée en tout temps pour reconstruire votre plateforme applicative. La sauvegarde crée un fichier de type ZIP file qui est automatiquement envoyé a l'adresse mail du compte  gmail de l'ASBV de même qu'a l'administrateur du système. Vous pouvez a volonté créer une sauvegarde en utilisant la fonction sauvegarde de la page d'accueil. Le nom du  fichier de sauvegarde contient la date et l'heure (  Exemple  : backup-2011-01-14 17-03-18.zip ). Cette fonction n'est disponible que lorsque vous êtes identifies et reconnu comme administrateur de filiale  ( Voir le profil du transporteur )

Une autre fonction permet de  restaurer la partie applicative et la base de données des transports a partir d'un fichier ZIP crée par la fonction sauvegarde. Cette action va automatiquement mettre a jour les données et le code de l'application tel qu'il était  a  l'heure de la création de la sauvegarde.  Toutes les données modifiées depuis cette  date seront  perdues.
