Démarrage du système

La  procédure suivante dot être suivie pour le démarrage de l'application de gestion de transport.

Etapes pour la mise en route

Description ou contexte d'utilisation

1.-  Mettre sous tension le  router Netgear 3G Interrupteur noir sur la face arrière droite du  routeur
2.-  Mette sous tension la tour PC  Acer Interrupteur  sur le sommet avant de la tour . Appuyer sur l'interrupteur  général
3.-  Vérifier que l'écran LCD est enclenché Interrupteur sur le bas de l'écran a droite ( La lumière doit  être ambrée  )
4.- Laisser à Windows le temps de démarrer Attendre de voir le bureau de Windows sur l'écran
5.- Cliquer sur l' icone  TransportASBV Attendre la page de Login de l'application
6.-Utiliser votre identification et mot de passe 1ere lettre de votre prénom suivie de votre  nom de famille. Utiliser le mot de passe  qui vous a été assigné ou celui que vous avez défini
7.- Vérifier sur la clef USB 3G ( Connexion réseau ) que vous n'avez aucune lumière qui clignote. Clef USB vers la fenêtre. Regardez le dos de la clef et chercher un indicateur lumineux bleu ou violet qui confirme le  bon accès a Internet
8.- Attendez la page d'accueil de l'application Le système doit créer avant le démarrage de l'application une copie de sécurité ..  soyez patients
9.-  Commencer  votre travail de permanencier En cas de doute utiliser les pages mode d'emploi 
10.- En cas de problèmes contacter votre support
