Extraction et Normalisation utilisées pour la suggestion de  chauffeur 

La liste des chauffeurs disponibles est soumise a un nouveau tri pour améliorer la finesse du choix des chauffeurs. Les  informations suivantes vont être extraites de la base de données et de Google Map

  • La distance entre la résidence du passager et celle du bénéficiaire. Il s'agit la d'éviter au chauffeur un trop long  trajet pour rejoindre son client. Les  chauffeurs  a proximité du passager seront  favorisé.
  • Le nombre de fois que ce chauffeur a déjà transporte ce passager.  On veut essayer d'assigner dans la mesure du possible le même chauffeur
  • On vérifie si le chauffeur a déjà un autre transport  pour le même jour.  On veut essayer d'éviter de mobiliser le chauffeur  plusieurs fois dans la même journée.
  • On vérifie  le nombre de trajets effectués par ce chauffeur dans les 15 derniers jours.  L'idée  ici est de  répartir les transports sur les différents chauffeurs pour garder un minimum d'engagement du bénévole. Un chauffeur qui a déjà  effectué de nombreux transports doit laisser un peu la place a d'autres candidats !
  • Nombre de Kilomètres parcourus par ce chauffeur dans les 15 derniers jours. Comme précédemment on cherche a répartir dans la mesure du possible les kilomètres sur le pool de chauffeurs.
  • On vérifie si le chauffeur a exprimé le désir de faire  que des transports locaux ( moins de  10 Km )

 

Pour chaque critère on associe une forme de note  do 0 a  100 . De cette facon un  chauffeur qui est dans la meme ville que son passager aura pour ce critere une note maximum. De la meme facon si un chauffeur n'a plus transporté de passager pendant 15 jours il se voit attribuer une note plus haute que celui qui a été sollicité pour plusieurs trasnports durant ce meme intervalle.  Ainsi pour chaque critère un score numérique est calculé.

 

 

A ce stade  on obtient une liste normalisée des chauffeurs  pour ce transport et il faut maintenant pondérer les différentes notes pour la qualification finale  Pour de plus amples informations sur le tri des chauffeurs accédez les  pages spécifiques de "Suggestion de Chauffeur " dans la section Mode d'emploi  ( Rubrique Moyenne Pondérées )
